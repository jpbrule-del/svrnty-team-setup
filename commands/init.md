# svrnty init — Initialize Orchestration

Initialize Overstory agent orchestration in the current project (Overstory + Beads + Mulch + worktrees).

## Usage

Run the init script. Pass any user-provided domain arguments through:

```bash
bash "${CLAUDE_PLUGIN_ROOT}/scripts/init-project.sh" $ARGUMENTS
```

Run this single command and present the output to the user. Do NOT run any other commands — the script handles everything.
